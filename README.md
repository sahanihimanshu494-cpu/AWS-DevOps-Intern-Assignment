# AWS DevOps Engineer Assignment

## Objective
Deploy a simple website on an AWS EC2 Ubuntu instance using Nginx.

## Task 1: EC2 Setup
- Created Ubuntu EC2 Instance
- Created Security Group
- Allowed Port 22 (SSH)
- Allowed Port 80 (HTTP)
- Connected remotely using PuTTY

## Task 2: Linux Basics
Commands Used:
```bash
sudo apt update
sudo apt install nginx -y
sudo systemctl status nginx
sudo systemctl restart nginx
df -h
free -h
ps aux
```

## Task 3: Website Hosting
- Created a simple HTML page.
- Replaced the default Nginx page.
- Hosted the website successfully.

## AWS Services Used
- Amazon EC2
- Security Group
- Key Pair

## Learning
- AWS EC2 instance creation
- SSH remote login using PuTTY
- Linux basic commands
- Nginx installation and management
- Hosting a static website
